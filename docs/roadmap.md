# 📈 stock-market-research 업그레이드 로드맵

> 작성일: 2026-04-22  
> 기준 버전: v2.0.0  
> 목적: 경제 지표 기반 **실전 투자 판단**을 지원하는 플러그인으로 진화

---

## 현재 상태 분석 (v2.0.0의 문제점)

### 🔴 카카오톡 — 너무 많고, 액션이 없다

| 현재 | 목표 |
|---|---|
| 9개 메시지 | 2~3개 메시지 |
| 데이터 나열 | 오늘 뭘 해야 하는지 한눈에 |
| 섹션별 분리 (거시/한국/미국/…) | 핵심만 압축, 투자 판단 중심 |

현재 메시지는 '보고서 목차'처럼 구성되어 있어 읽기 부담이 크다.  
카톡은 **"오늘 시장 방향 + 주목 이슈 1~2개 + 내일 변수"** 로만 충분하다.

---

### 🔴 문서 — 데이터는 많지만 "의미"가 없다

| 현재 | 목표 |
|---|---|
| 수치 나열 위주 | 수치 + 투자 해석 + 신호 |
| 섹션마다 표 한 개 | 섹션 첫머리에 "핵심 1줄 + 신호등(🟢🟡🔴)" |
| 고정된 섹션 순서 | 오늘 가장 중요한 이슈가 위로 |
| 전일 대비만 표시 | 전일·전주·전월 추세 비교 |
| 투자 판단 근거 없음 | 매수 유리 / 중립 / 매도 유의 신호 명시 |

---

### 🟡 투자 지원 기능 — 현재 전무

- 시장 전반 Risk Level 표시 없음
- 내 보유 종목과 오늘 뉴스 연결 없음
- 과거 데이터와 비교 없음 (오늘이 역사적으로 높은가 낮은가)
- 자동 실행 없음 (매일 수동 요청 필요)

## Phase 2 — v2.2.0: 문서 가독성 혁신

**브랜치:** `feat/22-doc-readability`  
**목표:** 보고서를 "읽어서" 투자 판단할 수 있는 수준으로

### 섹션별 신호등 헤더

모든 주요 섹션 첫머리에 아래 형식 추가:

```
┌─────────────────────────────────────────┐
│ 🟡 한국 증시 — 단기 중립 (관망 추천)      │
│ 외국인 4일 연속 순매도, 코스피 200일선    │
│ 이탈 여부가 이번 주 핵심                 │
└─────────────────────────────────────────┘
```

### 수치에 컨텍스트 추가

| 현재 표시 | 개선 |
|---|---|
| 원달러 1,467원 ▲0.6% | 원달러 1,467원 ▲0.6% (52주 최고 수준 — 수입 기업 주의) |
| VIX 18.5 | VIX 18.5 🟢 안정 (20 이하 = 시장 공포 없음) |
| CPI 118.80 | CPI 118.80 — 전월比 +0.3% (목표치 2% 대비 여전히 높음) |

### 추세 비교 열 추가

지수·환율 테이블에 "전일", "전주", "전월" 비교 열 추가:

| 지수 | 현재 | 전일 | 전주比 | 전월比 |
|---|---|---|---|---|
| 코스피 | 2,587 ▼0.5% | 2,600 | -1.2% | +3.4% |

→ "오늘 하락이지만 한 달 기준으로는 상승" 같은 맥락 제공

### 섹션 중요도 자동 정렬

오늘 가장 임팩트가 큰 섹션이 위로 올라오도록 동적 정렬:
- 지정학 이슈 high → 거시 정세 섹션 맨 위
- 외국인 수급 급변 → 수급 섹션 2번째로 이동

## Phase 5 — v3.0.0: 자동화 + 이력 관리 (MAJOR)

**브랜치:** `breaking/v3-automation`  
**목표:** 매일 장마감 후 자동 실행, 데이터 누적, 추세 분석

### 구조 변경 (MAJOR 이유)

1. **자동 실행** — Windows 작업 스케줄러 또는 Claude Code Cron으로 매일 오후 4시 자동 실행
2. **로컬 DB** — `data/history/YYYYMMDD.json` 형식으로 일별 수집 데이터 보존
3. **추세 분석 엔진** — 과거 30일 데이터 기반으로 "오늘이 얼마나 이례적인가" 계산
4. **알림 고도화** — 내 종목 5% 이상 급등락 시 즉시 카톡 알림 (일반 보고서와 별도)

### 추세 분석 예시

```
코스피 현재 2,587 — 최근 30일 평균 대비 -3.2% (하위 25th percentile)
원달러 현재 1,467 — 최근 30일 최고치 근접 (상위 90th percentile) ⚠️
VIX 18.5 — 최근 30일 평균과 유사 (정상 범위)
```
