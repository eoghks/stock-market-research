# v2.4.0 변경 사항 (2026-04-29)

## 기능 변경 (MINOR)

### Claude in Chrome 제거 — Yahoo Finance API + WebFetch 전환

**배경:** 기존 서브에이전트 A가 Claude in Chrome을 사용해 브라우저를 자동 제어했기 때문에,
사용자가 PC를 사용 중일 때 브라우저 창이 점유되는 불편함이 있었음.

**변경:**
- 서브에이전트 A 수집 방식을 **Yahoo Finance v7 API + WebFetch + WebSearch** 로 완전 교체
- 브라우저(Chrome) 불필요 — 사용자 화면을 건드리지 않고 백그라운드에서 실행
- `SKILL.md` 아키텍처 다이어그램 및 description 업데이트

**새 수집 방식:**

| 단계 | 방법 | 대상 |
|---|---|---|
| 1단계 | Yahoo Finance API 배치 쿼리 (단일 WebFetch) | 지수·환율·심리지표·섹터 ETF·종목 시세 전체 |
| 2단계 | WebFetch (hankyung.com) | 한국·미국·거시 뉴스 헤드라인 + URL |
| 2단계 | WebSearch | 기업별 최신 뉴스 (kr/us top10) |
| 2단계 | WebFetch (markets.hankyung.com) | 한국 섹터 등락률, 수급 동향 |
| 2단계 | WebSearch | 이벤트 캘린더 (FOMC·CPI·금통위·실적) |
| 3단계 | 자체 분석 | daily_insight·company_overall_summary·watchlist |

**Yahoo Finance API 배치 심볼 (45개, 인증 불필요):**
- 지수 12개: `^KS11,^KQ11,^GSPC,^IXIC,^DJI,^NDX,^SOX,^VIX,^N225,^HSI,000001.SS,^GDAXI`
- 환율 5개: `KRW=X,JPYKRW=X,CNYKRW=X,EURKRW=X,HKDKRW=X`
- 심리지표 4개: `^TNX,GC=F,CL=F,BTC-USD`
- 미국 섹터 ETF 8개: `XLK,XLF,XLE,XLV,XLY,XLI,XLB,XLP`
- 한국 시총 10위: `005930.KS ~ 055550.KS`
- 미국 시총 10위: `AAPL,MSFT,NVDA,AMZN,GOOGL,META,TSLA,BRK-B,AVGO,JPM`

**출력 JSON 스키마:** 변경 없음 (`chrome_data.json` 파일명 유지)
