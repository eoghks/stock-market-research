# v2.4.2 변경 사항 (2026-04-29)

## 기능 추가 (PATCH)

### Windows 작업 스케줄러 자동화

**배경:** Cowork 샌드박스는 외부 인터넷(DNS 포함)이 완전 차단되어 이메일 발송 불가.
로컬 Windows PC에서 실행 시 SendGrid 이메일 정상 작동. Windows 작업 스케줄러로
매일 자동 실행하여 Cowork 없이도 이메일 발송까지 완전 자동화.

**추가 파일:**
- `scripts/run_daily_report.ps1`: Claude Code CLI를 비대화형으로 호출하는 래퍼. 실행 로그를 `ai/stock/schedule_log.txt`에 기록.
- `scripts/setup_scheduler.ps1`: 1회 실행으로 작업 스케줄러에 "증시보고서_자동생성" 등록. 매일 18:30 KST 자동 실행, `StartWhenAvailable`로 PC가 꺼져 있어도 재시도.
- `docs/usage/scheduler-setup.md`: 설치 가이드, 실행 시각 변경, 수동 테스트, 파이프라인 흐름 도식.

**업데이트 문서:**
- `docs/architecture/pipeline.md`: 자동화 옵션 섹션 추가
- `README.md`: "자동 실행" 섹션 추가
- `SKILL.md`: Cowork 이메일 제한 및 로컬 자동화 안내 추가
