# v2.3.0 변경 사항 (2026-04-22)

## 신규 기능 (MINOR)

### 투자 신호 대시보드 페이지 (signalDashboard)

AI 인사이트 박스 다음, 섹션 0(거시 정세) 앞에 **전용 대시보드 페이지** 삽입.

#### 종합 신호 배너
```
🟢  오늘의 종합 투자 신호: 매수 유리
→ 신규 매수 진입 검토 — 분할 매수로 리스크 관리
```
- 5개 지표 합산으로 🟢🟡🔴 자동 판정
- 배너 배경색·테두리색이 신호에 따라 변경

#### 5개 지표 상세 테이블

| 지표 | 현재 신호 | 투자 시사점 |
|---|---|---|
| VIX (공포지수) | 🟢 18.5 | VIX < 18 → 공포 없음, 위험자산 선호 |
| 외국인 코스피 수급 | 🔴 -2,000억 | 외국인 순매도 → 하방 압력 주의 |
| USD/KRW 환율 | 🔴 1,467원 | 1,430원 초과 → 원화 약세 고점 경계 |
| 美 10년 국채금리 | 🟡 4.35% | 4.0~4.5% → 금리 부담 중간 수준 |
| 코스피 방향 | 🔴 ▼0.5% | 하락 마감 → 반등 확인 후 진입 |

#### 사용법 안내 박스
- 🟢 5개 → 적극 매수
- 🔴 3개 이상 → 신규 진입 자제
- 혼조 → 분할 매수로 리스크 관리
