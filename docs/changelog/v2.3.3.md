# v2.3.3 변경 사항 (2026-04-28)

## 버그 수정 (PATCH)

### PDF 한글 깨짐 수정

**원인:** `generate_report.js`가 `font:'Arial'`을 사용 중이었고, Arial은 한글 글리프가 없어
LibreOffice headless PDF 변환 시 한글이 □□□로 표시됨.

**수정:**
- `generate_report.js`: `const FONT = 'Malgun Gothic'` 상수 추가, 전체 18곳 `font:'Arial'` → `font:FONT` 교체
- `convert_to_pdf.js`: LibreOffice 변환 명령에 아래 옵션 추가
  - `--norestore`: 이전 세션 복구 대화 방지
  - `-env:UserInstallation`: 임시 프로필 디렉토리 지정 (충돌 방지)
  - `EmbedStandardFonts=true`: PDF에 폰트 임베딩 → 뷰어 환경 무관하게 정상 표시

### 이메일 PowerShell fallback 추가

**배경:** Cowork·샌드박스 환경에서 외부 SMTP 포트가 차단되어 nodemailer 발송 불가.

**수정:**
- `generate_report.js`: nodemailer 실패 시 `send_report_email.ps1` 자동 호출
  ```
  nodemailer 시도 → 실패 → PowerShell 스크립트 fallback → 성공 시 파일 삭제
  ```
- `send_report_email.ps1` (ai/stock/): `-PdfPath`, `-DocxPath` 파라미터 추가
  - 파라미터 지정 시: 해당 파일 직접 첨부
  - 파라미터 없이 실행 시: 오늘 날짜 기준 자동 검색 (기존 동작 유지)
