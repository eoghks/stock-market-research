# v2.2.0 변경 사항 (2026-04-22)

## 신규 기능 (MINOR)

### 섹션 투자 신호 헤더 (signalHeader)

주요 섹션(시장 심리·한국 증시·미국 증시·환율) 상단에 **투자 신호 헤더 박스** 추가.

```
🟢 한국 증시                          ← 신호등
코스피 2,587 (▲0.5%)  |  외국인 +1,200억   ← 핵심 수치 한 줄
→ 상승 추세 확인 — 신규 진입 검토 가능     ← 투자 행동 힌트
```

- 신호 색상: 🟢 초록(매수 유리) / 🟡 노랑(중립·관망) / 🔴 빨강(매도 유의)
- 섹션별 맞춤 행동 힌트 3종 (초록·노랑·빨강 각각)

### 투자 신호 자동 계산 (_SEC_SIGNAL)

보고서 생성 시 5개 지표를 자동으로 평가해 각 섹션 신호를 결정:

| 지표 | 🟢 | 🟡 | 🔴 |
|---|---|---|---|
| VIX | <18 | 18~25 | >25 |
| 외국인 코스피 수급 | 순매수(+) | 미수집 | 순매도(-) |
| USD/KRW | <1,350 | 1,350~1,430 | >1,430 |
| 美 10년 국채금리 | <4.0% | 4.0~4.5% | >4.5% |
| 코스피 방향 | ▲ | 보합 | ▼ |

### 섹션별 적용 범위

- **0-A. 시장 심리 패널** — VIX 기반 신호
- **1. 한국 증시** — 코스피 방향 + 외국인 수급 기반
- **2. 미국 증시** — S&P500 방향 기반
- **4. 환율 현황** — USD/KRW 레벨 기반 + 고점 경계권 문구 자동 추가
