# v2.0.0 변경 사항 (2026-04-22)

## 신규 기능

### 데이터·분석 섹션
- **Phase 9** — `0. 글로벌 거시 정세` 섹션 신설 (금리·지정학·에너지 헤드라인 우선 출력)
- **Phase 10** — 한국 업종별·미국 SPDR ETF 섹터 히트맵 (색상 강도로 자금 흐름 시각화)
- **Phase 11** — 외국인·기관·개인 수급 동향 + 외국인 순매수 상위/하위 5종목
- **Phase 12** — `0-A. 시장 심리 패널` (VIX·미10년물·WTI·금·BTC 신호 표시)
- **Phase 13** — `8-A. 이번주 주요 이벤트` 캘린더 (FOMC·CPI·금통위·실적발표)
- **Phase 14** — `8. 워치리스트` (거래대금급증·외인매수·급등락 기준 주목 종목 한국5+미국5)
- **Phase 15** — AI 데일리 인사이트 박스 (목차 직후, 데이터 기반 핵심 3줄)

### 시각화
- **Phase 8** — 시계열 차트 PNG (chartjs-node-canvas): 일7일·주4주·월12개월 스케줄
  - 코스피·코스닥·S&P500·나스닥·USD/KRW 라인차트 섹션별 삽입

### 보고서 품질

## JSON 스키마 신규 필드

```
daily_insight     — AI 핵심 3줄 배열
event_calendar    — 이벤트 캘린더 (date/region/event/impact/note)
watchlist         — {kr: [...], us: [...]} 워치리스트
flow_data         — 수급 동향 (kospi/kosdaq + top_buys/top_sells)
sentiment         — 심리 패널 (name/value/change/signal)
kr_sectors        — 한국 섹터별 등락률
us_sectors        — 미국 SPDR ETF 등락률
macro_headlines   — 거시 뉴스 (title/summary/url/category/importance)
timeseries        — 시계열 데이터 (KOSPI_daily 등)
kr_news[].url     — 뉴스 URL (v1에서 단순 문자열 → {title,url} 객체로 변경)
us_news[].url     — 동일
kr_top10[].news[].url — 동일
us_top10[].news[].url — 동일
```
